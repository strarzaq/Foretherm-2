► Google Sheets Integration — Setup Instructions
1. Create a new Google Sheet with this header row in row 1:
timestamp | site | date | measured_2m | height_offset | estimated_cable | forecast_mean | forecast_cold | forecast_warm | error_2m | error_cable | notes
2. In the sheet, go to Extensions → Apps Script and paste this code:
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.timestamp, data.site, data.date, data.measured_2m, data.height_offset, data.estimated_cable, data.forecast_mean, data.forecast_cold, data.forecast_warm, data.error_2m, data.error_cable, data.notes ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
3. Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and copy the URL.

⚙ Height Offsets — Station to Cable Level

Enter the vertical drop in metres from your weather station (2m measurement height) down to the cable/cordon position. Negative values = cable is lower = colder. This is added to the site correction factor.
